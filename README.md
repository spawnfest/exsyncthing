# ExSyncThing

Pure Elixir implementation for SyncThing.

Components:
- Syncthing Client
- Web Client for individual file up/download
- Local Discovery Server
- [todo] Global Discovery Server (not implemented until main client is working safely)
- [todo] Relay Server
